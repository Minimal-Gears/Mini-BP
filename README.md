# Mini-BP

Mini-BP is a small project to handle business processes with an internal mechanism to define workflows and create task inbox (cartable).

## Features

- **Workflow Definition**: Define business processes using an intuitive internal mechanism
- **Task Management**: Centralized task inbox (Cartable) system for managing and organizing workflow tasks
- **Process Execution**: Execute workflows with built-in state management and process tracking
- **Lightweight Design**: Minimal dependencies focused on core BPMS functionality

## Technology

Built with C# for easy integration into existing .NET applications.

## Use Cases

Mini-BP is suitable for:
- Small to medium-sized projects requiring workflow automation
- Custom business process solutions with fine-grained control
- Integration into C# applications needing BPM capabilities
- Learning BPMS concepts and implementation
